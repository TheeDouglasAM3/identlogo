# IDENTLOGO

# Demonstração da aplicação
O projeto desenvolvido neste repositório está disponível em: 
[https://identlogo-douglasalvesmarcelino.vercel.app/](https://identlogo-douglasalvesmarcelino.vercel.app/)

# Tecnologias
Neste projeto foram utilizadas as seguintes tecnologias:
* [React](https://pt-br.reactjs.org/)
* [TypeScript](https://www.typescriptlang.org/)

# Como rodar o projeto
```bash
# Copie o repositório
$ git clone https://github.com/TheeDouglasAM3/identlogo.git

# Instale as depedencias
$ npm install
ou
$ yarn install

# Rode a aplicação
$ npm start
ou
$ yarn start
```
Acesse o website em: http://localhost:3000/
